# Tarea 3 – Optuna y Regularización

Repositorio con la implementación de una red neuronal densa para clasificación de dígitos usando el dataset MNIST.

## Contenido

- Notebook con experimentos usando Optuna
- Aplicación de regularización:
  - L1
  - L2
  - L1-L2
  - Dropout
  - Dropout + L1-L2

## Herramientas usadas

- Python
- TensorFlow / Keras
- Optuna
- Google Colab
